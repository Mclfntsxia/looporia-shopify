# 🌸 Looporia Co.

A modern e-commerce website designed for Looporia Co., a handmade floral brand specializing in fuzzy wire flower arrangements. The website was created to showcase products with a clean, elegant, and user-friendly shopping experience.

## Project Overview

Looporia Co. is a concept online store that highlights handcrafted fuzzy wire flowers suitable for gifts, home décor, offices, and special occasions.

## Features

- Responsive website design
- Modern and minimalist layout
- Product showcase
- Collection pages
- Shopping cart
- Mobile-friendly interface
- Clean navigation
- Professional branding

## Platform & Tools

- Shopify
- Canva
- UI/UX Design
